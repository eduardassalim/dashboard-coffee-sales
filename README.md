# Coffee Sales Dashboard ☕ 
## Sobre o Projeto
Este dashboard foi desenvolvido para analisar o desempenho de vendas de uma cafeteria ao longo de um ano. O objetivo é fornecer visibilidade sobre o faturamento, volume de transações, ticket médio e comportamento do consumidor para apoiar a tomada de decisão estratégica.

## Principais Métricas (KPIs)
Total Revenue: Faturamento bruto acumulado.
Average Ticket: Valor médio gasto por pedido.
Total Sales: Volume total de pedidos realizados.

## Insights Extraídos
A partir da análise dos dados, foram identificados pontos-chave para o negócio:
Pico de Vendas: As terças-feiras apresentam o maior volume de transações.
Sazonalidade: Março foi o mês de maior faturamento no período analisado.
Preferência do Cliente: O "Americano with Milk" é o produto mais vendido, seguido pelo "Latte".
Período Crítico: O período da tarde concentra a maior parte do faturamento (33.97%).

## Tecnologias Utilizadas
Power BI / Excel: Para criação das visualizações e modelagem dos dados.
DAX (Data Analysis Expressions): Para cálculos de medidas personalizadas e inteligência de tempo.
Tratamento de Dados: Limpeza e organização de datas para ordenação cronológica.
